# 🤖 BigQuery AI Monitor

AI-powered pipeline monitoring using Google Gemini
that automatically analyzes the ecommerce BigQuery
pipeline and generates intelligent health reports.

## What It Does
→ Connects to BigQuery ecommerce tables
→ Collects key metrics automatically
→ Sends to Gemini AI for analysis
→ Generates detailed health report
→ Flags anomalies and issues
→ Provides specific recommendations

## Real Detection Example
Detected 55.1% drop in daily orders
compared to yesterday and recommended
immediate investigation!

## Tech Stack
→ Google Gemini API (FREE)
→ BigQuery
→ Python
→ GitHub Actions
